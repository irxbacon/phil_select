About This Tool
How It Works

This web application replicates the complex scoring algorithms from the original Philmont Excel spreadsheet. It considers multiple factors:

  • Program Scores: Individual activity preferences weighted by a program factor
  • Crew Preferences: Region, difficulty, and special feature preferences
  • Altitude Factors: Maximum altitude and daily elevation change considerations
  • Aggregation Methods: Total, Average, Median, or Mode scoring
